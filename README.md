# django-simple-test
Just a test for django

## How to run:

### Usin Docker:

`sudo docker build . --tag simple-django:test`

`sudo docker run -d -p 8080:80 --name ckll -t simple-django:test`


### Running in local machine

`python3 -m venv ./venv`
`source ./venv/bin/activate`
`cd simple-django-test`
